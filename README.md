# Flash-CHeck
A Flask based Invoice Generator

## Project Overview

This is a simple Invoice Management System built with **Flask**, which allows users to input details about items, including the name, quantity, price per item, and applicable tax rates. The system then generates an invoice, calculates the total cost, total tax, and displays a summary of the invoice.

## Key Features

- **Add Multiple Items**: Users can add multiple products to the invoice.
- **Tax Calculation**: Users can specify a tax rate for each item, and the system will calculate the tax for each item.
- **Dynamic Invoice Generation**: After submitting the form, the total price and tax are calculated and displayed in an invoice.

## Technologies Used

- **Flask**: Python web framework used for handling backend logic.
- **HTML/CSS**: Used for the front-end user interface.
- **JavaScript**: Used for dynamically adding items to the form.

## Unique feature added part to this project

 - Claculates the Tax as well along with the price of the product.
